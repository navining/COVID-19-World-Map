Navi Ning (xning5), Suk Min Hwang (shwang53)

About this project..

1. A interactive data visualization on how severe COVID-19 is in each country with d3.js.

2. The graph is supposed to be a world map, in which each country has a color.

3. The more severe the epidemic is, the redder the color is.

4. Interactive dashboard where users can filter the shown data on the map
